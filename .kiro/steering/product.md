# Product Overview

IIDX Tier Tracker is a web application for tracking clear lamp progress on IIDX (beatmania IIDX) tier lists. It helps rhythm game players manage and visualize their skill progression across difficulty tiers.

## Core Capabilities

1. **Lamp Management** - Track clear lamp status (NO PLAY, FAILED, EASY CLEAR, CLEAR, HARD CLEAR, FULLCOMBO) for songs across difficulty tiers
2. **Progress Visualization** - Dashboard view showing tier completion rates and lamp distribution
3. **Statistics Tracking** - Historical lamp progress, level-based summary statistics, and contribution calendar
4. **User Authentication** - Google OAuth-based user accounts for personalized tracking
5. **SP/DP Mode Support** - Track progress for both Single Play and Double Play modes

## Target Use Cases

- **Skill Progression Tracking**: Players monitoring their improvement across difficulty tiers
- **Goal Setting**: Identifying songs to work on for tier advancement
- **Historical Review**: Viewing progress over time via statistics and history

## Value Proposition

- **Tier List Integration**: Built around IIDX tier lists (地力表) for structured progression
- **Persistent Progress**: Cloud-saved lamp data with Google authentication
- **Clean UI**: Simple, focused interface for lamp management without unnecessary complexity
